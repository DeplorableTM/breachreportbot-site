How It Works

A quick guide to getting set up and using BreachReportBot.

1. Add the bot to your server

An admin invites BreachReportBot and picks a channel for it to post in. That's the only setup step required for the server itself.

  • Add BreachReportBot to Discord
  • /setchannel — pick the channel where leaderboards, records, and session recaps get posted (admin only)

2. Link your Ubisoft account

Each player links their own Ubisoft username to their Discord account. This is self-serve — an admin can't link an account on someone else's behalf. Linking an account requires an active subscription — either your own personal subscription or a seat assigned to you from your server's subscription bundle. If you don't have one yet, /addplayer will point you to a checkout link instead of linking the account.

Currently PC / Ubisoft Connect only — console accounts (Xbox, PlayStation) can't be tracked yet. Console support is planned.

  • /addplayer — link your Ubisoft username (requires an active subscription)
  • /myplayer — see which accounts you currently have linked
  • /removeplayer — unlink an account

Once linked, the bot automatically polls for new matches and updates your stats — no further action needed.

3. Check your stats

  • /6stats — your current ranked stats (or look up another player)
  • /sessionstats — stats for the current tracking session
  • /lifetimestats — your all-time stats
  • /seasonstats — your RP progress this season

4. Leaderboards & records

These post automatically as your server plays, or you can pull them up any time. Leave the mode blank to see every mode your server has enabled at once.

  • /leaderboard — tonight's leaderboard
  • /monthlyleader, /seasonleader, /lifetimeleaderboard
  • /monthlypersonalrecords, /seasonpersonalrecords — your own personal bests
  • /monthlyalltimerecords, /seasonalltimerecords — server-wide records

5. Customize how your server tracks stats (admin only)

  • /setmodes — turn Ranked, Quick Match, and Unranked tracking on or off
  • /setpoints — adjust how much each stat is worth for leaderboard scoring
  • /settimezone — set your server's timezone for daily rollovers
  • /scoring — see the current point values

Licenses & seats

Every linked Ubisoft account uses a license seat. Seats can be purchased as a personal subscription or as a server bundle (an admin buys a block of seats and assigns them to members).

  • /subscribe — get a checkout link to purchase a license, or use the Store button the bot shows you when you try to link an account without an available seat
  • /mylicenses — see every seat or grant you've purchased
  • /assignseat, /unassignseat — admins assign purchased server seats to members
  • /listseats — admins see every seat currently assigned in this server

Need the full list?

Run /help in Discord at any time to see every command available to you.
